A jQuery weboldala

A jQuery népszerű JavaScript könyvtár, mely a HTML kód és a kliensoldali JavaScript közötti kapcsolatot hangsúlyozza. 2006 januárjában jelentette meg a Mozilla Alapítvány népszerű JavaScript evangelistája, John Resig. A függvénykönyvtár MIT és GNU kettős licenc alatt jelent meg. A jQuery ingyenes, nyílt forrású szoftver.

Számos ismert IT cég is alkalmazza a jQuery-t saját projektjeiben, például a Microsoft erre építette a Visual Studióban is elérhető ASP.NET AJAX platformját.[2]

A jQuery célja, hogy segítsen minél inkább leválasztani a JavaScript kódot a HTML-ről, és kényelmes kommunikációt biztosítson a weblap elemeivel – eseményvezérlők és azonosítók (ún. CSS szelektorok) használatával.

Jellemzők[szerkesztés]

  • Sizzle JavaScript–CSS szelektor, a jQuery projekt egyik „spin-off”-ja[3]
  • Eseménykezelés
  • CSS manipuláció
  • Effektek és animációk kezelése
  • AJAX támogatás
  • JavaScript Pluginek

Új képjegyzet Jelölj ki egy téglalapot a fenti képen (a bal egérgomb nyomvatartásával). Ehhez a képhez jegyzetek tartoznak. Húzd az egeret a kép fölé a megjelenítésükhöz.  A jegyzetet a(z) X oldalon tudod szerkeszteni. Miért szeretnéd törölni a jegyzetet?


Mentés

Képjegyzet módosításához olyan böngészőre van szükség, ami támogatja a XMLHttpRequest objektumot. A te böngésződ nem támogatja ezt az objektumot, vagy nincs engedélyezve a használata (Internet Explorerben előfordulhat, hogy az ActiveX komponens ki van kapcsolva), így nem tudod módosítani a képannotációt. Elnézést a kellemetlenségért.
